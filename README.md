# scdoc

scdoc is a simple man page generator for POSIX systems written in C99.

## Installation

    make
    sudo make install

You can pass PREFIX or DESTDIR to make if you'd like:

    make PREFIX=/usr
    sudo make PREFIX=/usr install

Uninstallation is similar:

    sudo make uninstall

See scdoc(1)
